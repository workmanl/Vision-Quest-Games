# Vision Quest Games

A collection of fun, engaging games designed to help children strengthen their visual skills through play!

## Games Included

### Space Explorer
Catch colorful planets falling from space! Track multiple objects and improve visual attention. Includes Rocket Dodge bonus mode.

### Balloon Adventure
Pop balloons matching the target color! Develops color recognition and hand-eye coordination. Includes Keepy Uppy bonus game.

### Memory Master
Classic memory matching with beautiful cards. Strengthens visual memory and concentration. Includes Shell Game bonus mode.

### Nitro Snail Runner
An endless runner with triple jumps, power-ups, and 5 unique biomes to explore. Help our speedy snail jump over obstacles and collect gems!

### Treasure Maze
Guide a wizard through randomly generated mazes. Collect gems and find the exit! Includes Match-3 puzzle mode.

### Alchemy Lab
Combine elements to discover new creations! Start with Earth, Fire, Wind, and Water, then unlock hundreds of combinations.

## Features

- 6 unique games with multiple modes
- XP and leveling system
- 50+ badges to unlock
- Relaxing background music
- Kid-friendly design
- No ads
- No in-app purchases
- Works offline

## How to Play

1. Open the app and tap any game to start
2. Follow the on-screen instructions for each game
3. Earn XP and unlock badges as you play
4. Tap the question mark on the main screen for tips
